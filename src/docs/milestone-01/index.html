Overview

TransKnowledge is a carefully crafted web application that seeks to centralize information and resources for transgender people's wellbeing. It features a map that integrates the latest information on transgender safety laws with GPS routing to create road maps that prioritize trans safe havens; a news article aggregator for the latest in trans information; a moderated community space focused on outreach and mutual care; and an ever growing database of trans resources that anyone can contribute to.
